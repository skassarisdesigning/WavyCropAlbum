# Wavy Crop Album ✂️

Веб-приложение для фигурной обрезки изображений с волнистыми краями (эффект «фигурного дырокола») и размещения результатов в виртуальный альбом.

## Возможности

- **Загрузка изображений** — drag-and-drop или выбор файла
- **Фигурная обрезка** — прямоугольник с волнистыми краями (canvas clipping path)
- **Настройки** — ширина/высота обрезки, амплитуда и частота волны
- **Виртуальный альбом** — сетка обрезанных фото с возможностью удаления
- **Сохранение** — альбом сохраняется в `localStorage`

## Запуск

Просто откройте `index.html` в браузере. Никаких зависимостей или сборки не требуется.

```
start index.html
```
